.database().table()
Source:

This is the table class. With this class you can manage the table returned by the database.database.table function.

Examples

cerus.database("test").table("test_table");

.clear()
Source:
lib/database.js, Line 217
Access:
public
Summary:
Clears the table.

This function clears the table. This means removing everything that is in the table. Use database.database.table.drop to delete the whole table, instead of its contents. It returns a promise that is called when the table is cleared.

Examples

cerus.database("test").table("test_table").clear();
// -> clears everything in the table

Promise
Description
This function returns a promise.

.close()
Source:
lib/database.js, Line 201
Access:
public
Summary:
Closes the connection to the database.

With this function you can close the connection to the database. It is basically an alias of the database.database.close function. You should close connections when you're finished with them, to relieve stress from the system. It returns a promise that is called when the connection has been closed.

Examples

cerus.database("test").table("test_table").close();
// -> closes the connection to the database

Type
Promise
Description
This function returns a promise.

Summary:
Modifies the specfied rows.

With this function you can modify the specified rows. You can specify the rows using the query parameter. Multiple rows can be selected to be modified. The columns that will be modified are specified using the change paramater. This function will return a promise that is called when the rows have been modified.

Parameters

Name
Type
Description

The rows that will be modified.

Object
The changes that will be done to the rows.

Examples
