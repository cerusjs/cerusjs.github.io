This is the main class for the cerus module. It isn't really a class since it creates an instantiation of the core class and adds some plugins to the core. This class also adds a root function to cerus and some of the plugins that are added can also be found in this package.

.root(root)
Source:
lib/cerus.js, Line 18
Access:
public
Summary:
Returns the project's root location.

This is the setter and getter for the project's root location. By default this root is a fixed version of process.cwd(). The fix that is applied is replacing all '\' with '/' to add support all platforms. The default root also contains a slash at the end to let every platform know it is definitely a folder.

Parameters

Name
Type
Description
root
String
The new path for the root. (optional)

Examples

// with as project env "C:\\project\\example.js"
console.log(cerus.root());
// -> will return "C:/project/"

Returns

Type
String
Description
The root path.
