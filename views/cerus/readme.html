CerusJS is a open, pluginable and free to use framework for NodeJS. Since everything is based around plugins, it can be as lightweight as you want it to be. This is a "hub" module. It contains all the basic modules and plugins you need to create a basic webserver.

Installation

CerusJS is available through NPM. Before installing CerusJS make sure you have NodeJS installed.

# Installs the latest version of CerusJS.
$ npm install cerus

Plugins & Modules

  • The cerus-server module contains all the server related functions.
  • With cerus-router you can route requests based on their URL.
  • The cerus-api module is used for routing REST requests.
  • With the cerus-settings module you can easily create and use settings.
  • The cerus-promise module is a custom promise.
  • Use cerus-uuid for working with UUIDs.
  • The cerus-compression module is a compression utility.
  • With cerus-fs you can write to, read from and monitor files and directories.
  • Use cerus-sessions to control user-sessions.

Creating a Server

To create a basic server you only have to use the following code.

var cerus = require("cerus")();
cerus.server().start();
