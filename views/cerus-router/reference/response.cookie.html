response.cookie()
Source:
lib/response/cookie.js, Line 1
Access:
public

This is the cookie class. It is basically a model of a cookie, meaning that with this class you can edit a cookie that will be send to the client. To send it to client you do need to add it using response.cookies().add(cookie).

Examples

var cookie = response.cookies().create("key", "value");
cookie.path("/test");
response.cookies().add(cookie);

Summary:
The setter/getter for the same setting.

This is the setter and getter for the same setting. This setting sets if the cookie is only allowed to be send to the same site where it came from. There are two options for this: "lax" will allow the client to send the cookie to other sites and "strict" tells the client to only send the cookie to the set domain.
